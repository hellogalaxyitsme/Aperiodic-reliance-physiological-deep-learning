# Results Index

Aggregate outputs are stored under `results/tables/`. These files are small
enough to track in Git and provide the main numerical record of the analyses.

- `combined_intervention_summary.*`: cross-domain intervention summary.
- `full_sleep_edf_multiseed_subject_bootstrap.*`: Sleep-EDF multiseed neural
  results.
- `sleep_edf_validation_controls_bootstrap.*`: Sleep-EDF control and validation
  analyses.
- `tuab_full_multiseed_neural_subject_bootstrap.*`: full-TUAB neural results.
- `tuab_full_age_matched_multiseed_neural_subject_bootstrap.*`: full-TUAB
  age/sex-matched neural controls.
- `tuab_full_foundation_multiseed_subject_bootstrap.*`: seven-model TUAB
  foundation-model audit.
- `physionet_mi_multiseed_neural_subject_bootstrap.*`: PhysioNet MI neural
  audit.
- `ptbxl_ecg_architectures_prediction_bootstrap.*`: PTB-XL ECG architecture
  audit.
- `ptbxl_matched_ecg_architectures_prediction_bootstrap.*`: PTB-XL age/sex-
  matched neural audit.
- `formal_hypothesis_tests*`: bootstrap p-values and Benjamini-Hochberg FDR
  correction.
- `aperiodic_simulation_validation/`: synthetic ground-truth validation.
- `raw_intervention_diagnostics/`: sham reconstruction and distributional
  diagnostics.
